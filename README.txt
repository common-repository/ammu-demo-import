== Description ==

A plugin to install demo content to themes developed by Ammuthemes.
This plugin requires the [One Click Demo Import](https://wordpress.org/plugins/one-click-demo-import/) plugin to be installed.


== Supported Themes ==

1. [Real Estate Salient](https://wordpress.org/themes/real-estate-salient/)

== Installation ==

*** From your WordPress dashboard ***

1. Visit 'Plugins > Add New',
2. Search for 'Ammu Demo Import' and install the plugin.
3. Activate 'Ammu Demo Import' from your Plugins page.


== Frequently Asked Questions ==

= Does this plugin works with any WordPress theme =

No, This plugin will only work with the themes developed by [Ponvendhan](https://ammuthemes.com/).
